# 🌴 Lakshadweep Travel Booking Website

🔗 **Live Demo:** https://nirman-2-0.vercel.app/

A modern and responsive travel booking website for Lakshadweep built using HTML and CSS. This project provides users with an engaging interface to explore destinations and simulate a booking experience.

---

## ✨ Features

- 🏝️ Beautiful homepage with island-themed design  
- 📍 Destination sections showcasing Lakshadweep islands  
- 🧾 Booking form UI for travel reservations  
- 📱 Fully responsive (mobile + desktop friendly)  
- 🎨 Clean and aesthetic user interface  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- Vercel (for deployment)
